HTML битнең төп төзелеше
------------------------

```html
<!DOCTYPE html>
<!-- Бу комментари. Биттә берничек тә күрсәтелмиячәк. -->
<html>
  <head>
    <meta charset="utf-8">
    <title>This is page title (will be shown on browser tab title)</title>
  </head>
  <body>
    <!-- Битнең тәне. Төп контент монда урнаша -->
  </body>
</html>
```
